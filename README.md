## ▶️ Como executar os arquivos

1. Clone o repositório:

   ```bash
   git clone https://github.com/johnnykenneth/Exercicios_das_aulas_praticas.git
   ```

2. Acesse a pasta desejada:

   ```bash
   cd "Aula prática 01"
   ```

3. Compile o arquivo:

   ```bash
   javac NomeDoArquivo.java
   ```

4. Execute:

   ```bash
   java NomeDoArquivo
   ```
